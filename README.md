# 🎮 Tic Tac Toe

A responsive **Tic Tac Toe** game built with **HTML, CSS, and JavaScript**. It features two-player gameplay, automatic winner detection, draw detection, and a clean user interface.

## 🚀 Features

* Two-player gameplay
* Winner & draw detection
* New Game functionality
* Responsive design
* Clean and simple UI

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
